# 数据标准化规范

**目标**: 确保所有33个国家的数据维度一致

---

## 📋 统一标准

### 1. 年龄档位 (5档)
```
- 18-24 岁 (%)
- 25-34 岁 (%)
- 35-44 岁 (%)
- 45-54 岁 (%)
- 55+ 岁 (%)
```

### 2. 教育水平 (3档)
```
- 大学及以上 (%) [包括本科、硕士、博士]
- 大专/高等专科 (%)
- 高中及以下 (%) [包括高中、初中、小学]
```

### 3. 婚姻状况 (3档)
```
- 已婚 (%)
- 未婚/单身 (%)
- 其他 (%) [包括离异、丧偶、未知]
```

### 4. 性别比例 (2档)
```
- 男性 (%)
- 女性 (%)
```

### 5. 收入水平 (因国家而异)
```
各国按当地货币或购买力等价标准:
- 高收入 (%)
- 中高收入 (%)
- 中等收入 (%)
- 低收入 (%)
```
