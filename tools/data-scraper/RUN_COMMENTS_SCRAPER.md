## 关于 189 条 vs 100 条视频

### 为什么只有 100 条而不是 189 条？

根据 Apify Actor 日志分析：

- **TikTok 官方限制**: TikTok 一次性 API 请求可返回的视频数量有上限（~100 条）
- **Actor 内部限制**: Apify 的 `tiktok-profile-scraper` 有硬性限制，无法突破
- **日志显示**: "No more videos to scrape" 说明已到达技术上限
